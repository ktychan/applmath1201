Let's build some vocabulary for building simple models using scaling relations, c.f. Example~\ref{ex:toucan}.

\begin{definition}
  Suppose two quantities \(x,y\) are related by a power rule \(y = c x^{\alpha}\) for some constant \(c > 0\).
  \begin{itemize}
    \item If \(\alpha = 1\), then \(x\) and \(y\) exhibit isometry (proportional change).
    \item If \(\alpha \ne 1\), then \(x\) and \(y\) exhibit allometry (not proportional change).
  \end{itemize}
\end{definition}

\faStar{} The language ``increases by \(x\%\)'' and ``decreases by \(y\%\)'' \hlwarn{Edited: ``decrease'' \(\to\) ``decreases'' (both quoted phrases on this line)} show up a lot in science. Remember that ``increases by \(x\%\)'' means to scale by \(1+x/100\) and that ``decreases by \(y\%\)'' means to scale by \(1 - y/100\).  For example, a \(5\%\) increase means to scale by \(1.05\), and a \(1\%\) decrease means to scale by \(0.99\).

Another way to remember this language is to think about wage. A \(50\%\) increase in wage is \(1.5\) times of the usual pay, like statutory holiday pay. A \(100\%\) increase in wage means double the pay!
