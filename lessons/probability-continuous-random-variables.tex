\begin{document} \section{Continuous random variables}

A random variable is a continuous random variable if it takes values on an interval. The random variable associating time (outcomes) to temperature at YYZ (values) is a continuous random variable. Continuous random variables are described by their PDF or CDF.

\begin{definition}[PDF and CDF]
  Let \(X\) be a continuous random variable.

  The \hlmain{probability density function} for \(X\) is a nonnegative continuous function \(f(x)\) so that
  \[
    \mathbb{P}(a \le X \le b)
    \quad\text{is}\quad
    \int_{a}^{b} f(x) \;dx.
  \]

  The \hlmain{cumulative distribution function} for \(X\) is a nondecreasing and nonnegative function so that
  \[
    \mathbb{P}(X \le x)
    \quad\text{is}\quad
    F(x).
  \]

\end{definition}

Like discrete random variables, continuous random variables have expectation, variance and standard deviation.

\begin{definition}
  Suppose \(X\) is a continuous random variable with PDF \(f(x)\). \hlwarn{Edited: ``PMF'' \(\to\) ``PDF''; added the missing ``\(\;dx\)'' to both integrals; ``\(V(x)\)'' \(\to\) ``\(V(X)\)''} \hlwarn{FIXME: verify this math/content correction}
  \begin{enumerate}[itemsep={1ex}]
    \item Its expectation is \(\mu = E(X) = \int_{-\infty}^{\infty} x f(x) \;dx\).
    \item Its variance is \(V(X) = E\left( (X - \mu)^{2} \right) = \int_{-\infty}^{\infty} (x - \mu)^{2} f(x) \;dx\).
    \item Its standard deviation is \(\sigma = \sqrt{V(X)}\).
  \end{enumerate}

  Variance is often denoted as \(\sigma^{2}\).
\end{definition}

Expectation measures the \hlmain{centre} of data. Variance and standard deviation measure how far the data spread from its expectation.

We are moving closer to simulation. Sometimes (in reality and in this course), we want to randomly generate data that replicate experiments with certain sample mean and sample variance.

\begin{definition}[uniform distribution]
  The uniform distribution \(U(a,b)\) is defined by its probability density function
  \[
    f(x) = 
    \begin{cases}
      \frac{1}{b-a}, &\text{ if } a \le x \le b \\[2ex]
      0,            &\text{ otherwise }.
    \end{cases}
  \]

  Its expectation is \(\mu = \frac{1}{2}(a + b)\) \hlwarn{Edited: ``\(\frac{1}{2}(b - a)\)'' \(\to\) ``\(\frac{1}{2}(a + b)\)'' --- the mean of \(U(a,b)\) is \((a+b)/2\)} \hlwarn{FIXME: verify this math/content correction} and variance is \(\frac{1}{12}(b - a)^{2}\).
\end{definition}

Depending on the capabilities of your software and the requirements of your simulation/experiment, you may need to do a little conversion between specifying a uniform distribution by an interval \(U(a,b)\) or by its expectation and standard deviation.

\begin{method}
  The uniform distribution \(U(-c,c)\) has expectation and standard deviation
  \[
    \mu = 0
    \quad\text{and}\quad
    \sigma = \frac{c}{\sqrt{3}}.
  \]

  Conversely, if you need a uniform distribution with expectation \(\mu = 0\) and standard deviation \(\sigma\), then 
  \[
    c = \sigma \sqrt{3}
    \quad\text{with PDF}\quad
    f(x) = 
    \begin{cases}
      \frac{1}{2\sigma \sqrt{3}} &\text{if } -c \le x \le c, \\[2ex]
      0 &\text{otherwise}.
    \end{cases}
  \]
\end{method}
