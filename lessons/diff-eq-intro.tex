\begin{document} \section{Introduction to continuous-time models and differential equations}

\hlmain{Continuous-time models} are models in which time flows continuously as experienced by humans \hlwarn{Edited: ``human'' \(\to\) ``humans''}. In mathematical terms, the time variable \(t\) takes values over an interval. 

\hlmain{Differential equations} are equations for describing changes over time and are often used to describe continuous-time models.

We introduce these two ideas together through Example~\ref{ex:diff-eq-intro}.

\begin{example} \label{ex:diff-eq-intro}
  Let \(N\) measure the size of a population (e.g., cells, animals, any \emph{group} of organisms that live through some kind of life cycle). Assume time \(t\) \emph{flows continuously} and is measured in \emph{years}. Unlike discrete-time models, it makes sense to say, for example, \(0.23\) years.

  Assume that at all times, the rate of change at time \(t\) is proportional to the size of the population at time \(t\).  
  Express the rate of change of the population \hlmain{at time \(t\)} as an equation if the per capita birth and death rates \hlwarn{Edited: ``per capital'' \(\to\) ``per capita''; ``rate are'' \(\to\) ``rates are''} are \hlsupp{\(B = 10\%\) per year} and \hlsupp{\(D = 1\%\) per year}.
  Use dimensional homogeneity to stay on track.

  \emph{Per capita} means per unit of population, e.g., per cell, per \(10\) people, per unicorn {\emojifont{🦄}}, etc.  

  \blanks*
\end{example}

Let's sharpen the concept of differential equations by writing down a precise definition.
\begin{definition}
  Let \(u\) be a \emph{differentiable} function of \(t\) whose values are real numbers. A \hlmain{differential equation} (DE) is an equation that relates the function \(u\) to one or more of its derivatives.  The \hlsupp{unknown} in a differential equation is the function \(u\).

  An \hlmain{initial-value problem} (IVP) has two parts:
  \begin{enumerate}
    \item An \hlmain{initial condition} (or \hlmain{initial value}) is some known value \(u(\text{constant})\).
    \item A differential equation.
  \end{enumerate}

  \faExclamationTriangle{} Differential equations are used to describe continuous-time models.
\end{definition}

\begin{example}
  Here are two differential equations.  

  \begin{enumerate}[wide]
    \item A differential equation can be written with or without function notation.
      \[
        {\overbrace{N'(t) = r N(t) \left(1-\frac{N(t)}{K}\right)}^{\text{independent variable is shown}}} 
        \qquad\text{or}\qquad 
        {\overbrace{N' = rN\left(1-\frac{N}{K}\right)}^{\text{independent variable is hidden}}}.
      \]

      \blanks{10}

    \item An initial value problem is a differential equation with an initial condition.
      \[
        \overbrace{N(0) = 350}^{\text{initial condition}} \quad\text{and}\quad N'(t) = r N(t) \left(1-\frac{N(t)}{K}\right).
      \]
      \bigskip


    \item The initial condition can be measured at any time, often at time \(0\) but not always. The following is a valid IVP.
      \[
        \overbrace{N(-3.236492) = 473829}^{\text{a valid initial condition}} \quad\text{and}\quad N'(t) = r N(t) \left(1-\frac{N(t)}{K}\right).
      \]
      \bigskip

    \item A differential equation can involve any algebraic operations and have higher-order derivatives.

      Recall that \(v''\) and \(v^{(2)}\) are the notations for the second derivative of \(v\).
      \[
        v(t) = v'(t) + \frac{v''(t)}{t} + \frac{v^{(3)}(t)}{t^{2}} + \sqrt{t-1}
        \qquad\text{or}\qquad
        v = v' + \frac{v''}{t} + \frac{v^{(3)}}{t^{2}} + \sqrt{t-1}
      \]
  \end{enumerate}
\end{example}

A comparison of terminologies between discrete-time models and continuous-time models:

A recurrence has two parts:
\begin{enumerate}
  \item An initial condition, e.g., \(b_{0} = 1000\).
  \item A recurrence equation, e.g., \(b_{t} = 1.02 b_{t-1} - 50\).
\end{enumerate}

An initial-value problem also has two parts:
\begin{itemize}
  \item An initial condition, e.g., \(N(0) = 350\).
  \item A differential equation, e.g., \(N' = rN(1 - N/K)\). 
\end{itemize}

Please note that initial value problems are the continuous-time counterpart of recurrences for discrete-time models.
