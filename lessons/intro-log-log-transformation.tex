\begin{document} \section{Power rule models \texorpdfstring{\(y =  c x^{\alpha}\)}{} and log-log transformation}

Visualization is often a helpful method to \emph{make sense} of data collected from empirical studies.

\begin{definition}[log-log space]
  A point \((x,y)\) in the Euclidean space corresponds to \((\ln(x), \ln(y))\) in the log-log space, and a point \((\alpha,\beta)\) in the log-log space corresponds to \((e^{\alpha}, e^{\beta})\) in the Euclidean space.
\end{definition}

\begin{definition}[log-log transformation]
  The (natural) log-log transformation of an equation \(y = c x^{\alpha}\) is
  \[
    \ln(y) = \ln(c) + \alpha \ln(x)
  \]
  obtained by applying \(\ln(\cdots)\) to both sides of \(y = c x^{\alpha}\).
\end{definition}

\blanklines{8}

``Do taller people live longer?'' is a typical question for which a science-informed answer is about proportions. In this case, the exponent in \(y = c x^{\alpha}\) contains the important information, and the constant \(c\) has no effect (you'll see why in Example~\ref{ex:propto}). The \(\propto\) notation is often used.
\begin{definition} \label{ex:propto}
  If \(y = (\text{non-zero constant}) x^{\alpha}\), then we write \(y \propto x^{\alpha}\). 

  Conversely, if we write \(y \propto x^{\alpha}\), then \(y = (\text{non-zero constant}) x^{\alpha}\).
\end{definition}

\faStar{} The \(\propto\) notation dramatically simplifies calculations \hlmain{when we only care about proportions}.

\begin{example}
  If the radius of a sphere doubles, how does its volume change? Note \(V = \frac{4}{3} \pi r^{3}\).

  \blanklines{20}
\end{example}
