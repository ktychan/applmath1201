\begin{document} \section{Errors and probability}

Everyday science uses tests to make predictions. However, tests can be wrong. It is sensible to calculate and communicate how likely is a test wrong and in what way?

\begin{definition}\label{def:errors}
  Suppose a test returns \emph{either} positive \emph{or} negative for a condition. There are four possibilities, each with an associated probability, depending on whether the test result matches the reality.  

  If the test returns
  \begin{itemize}
    \item \textbf{negative} given the condition is \textbf{not present}, then the result is a \hlmain{true negative}.
      \[
      \text{specificity} = \mathbb{P}(\text{true negatives}) = 
      \mathbb{P}(\text{negative} \mid \text{not present})
      \]
    \item \textbf{positive} given the condition is \textbf{not present}, then the result is a \hlmain{false positive}, and the test admits a \hlmain{type-I error}.
      \[
      \mathbb{P}(\text{type-I errors}) = \mathbb{P}(\text{false positives}) = 
      \mathbb{P}(\text{positive} \mid \text{not present})
      \]
    \item \textbf{positive} given the condition is \textbf{present}, then the result is a \hlmain{true positive}.
      \[
      \text{sensitivity} = \mathbb{P}(\text{true positives}) = 
      \mathbb{P}(\text{positive} \mid \text{present})
      \]
    \item \textbf{negative} given the condition is \textbf{present}, then the result is a \hlmain{false negative}, and the test admits a \hlmain{type-II error}.
      \[
      \mathbb{P}(\text{type-II errors}) = \mathbb{P}(\text{false negatives}) = 
      \mathbb{P}(\text{negative} \mid \text{present})
      \]
  \end{itemize}

  \faStar{} True/false indicates whether the test result matches the reality.
\end{definition}
