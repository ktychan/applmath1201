\begin{document} \section{Solutions (plural!) of differential equations and initial-value problems}

In a differential equation, the \hlmain{dependent} variable, say \(u(t)\), is the \hlmain{unknown}. \emph{One} way to make sense of a differential equation is to try to \emph{solve it} --- express the unknown as down-to-earth functions (plural!) of the independent variable \(t\). 

\begin{definition}[solutions (plural!) of differential equations]
  Suppose we have a differential equation whose unknown is \(u(t)\).

  A function \(f(t)\) is \hlmain{a solution of the differential equation} if we can \hlsupp{produce an equality} \emph{after} replacing \(u(t)\) in the DE with \(f(t)\).  
\end{definition}

\faStar{} To verify a \emph{proposed} solution for a differential equation, we focus on the unknown function.

\begin{example} \label{eg:de-solution-intro}
  Which of the following functions are solutions to \hlwarn{Edited: removed dangling ``the''} \(u'(t) = t u(t)\)?
  \[
    f_{1}(t) = e^{t^{2}}, \qquad f_{2}(t) = 2e^{t^{2}/2}, \qquad f_{3}(t) = 3e^{t^{2}/2}.
  \]

  \blanks{35}
\end{example}
\clearpage

Here is a suggestion to manage a long computation especially if you tend to make little calculation mistakes.
\begin{method}
  Suppose we want to check if a proposed function \(f(t)\) is a solution to a given differential equation whose unknown is \(u(t)\).

  \textbf{Prepare}.
  \begin{enumerate}[itemsep={0ex}]
    \item Reorganize the DE to the form \hlmain{\((\cdots \text{stuff} \cdots) = 0\)}.
    \item Rewrite \(u'(t)\) as \(\frac{d}{dt} \bigg( u(t) \bigg)\) so that you know what to differentiate later.
    \item Discard the ``\hlmain{\(=0\)}'' on the right-hand side. 
  \end{enumerate}

  \textbf{Calculate}.
  \begin{enumerate}[itemsep={0ex}]
    \item Replace every \(u(t)\) with \(f(t)\) in \hlmain{\((\cdots \text{stuff} \cdots)\)}. 
    \item Perform all differentiation and simplify.
  \end{enumerate} 

  \textbf{Decide}. Assuming no mistake was made, if you get \(0\) at the end, then \(f(t)\) is a solution; otherwise, \(f(t)\) is not a solution.
\end{method}

\begin{exercise}[concept check]
  Which of the following is a solution to \(\frac{dy}{dt} - 2y = 2t - 1\)?
  \[
    f_{1}(t) = 3e^{2t} - 1, \qquad f_{2}(t) = 3e^{2t} - 2.
  \]

  \blanks{30}
\end{exercise}
\clearpage


\begin{definition}[Solutions to IVP]
  Suppose we have an initial-value problem whose initial condition is \(u(0)\).  

  A function \(f(t)\) is \emph{the} (one and only) solution to the IVP if 
  \begin{itemize}
    \item \(f(0) = u(0)\), and 
    \item \(f(t)\) is a solution to the differential equation.
  \end{itemize}
\end{definition}

Exercise~\ref{eg:de-solution-intro-ivp} is a sequel of Example~\ref{eg:de-solution-intro}.  The presence of an initial condition tells you this is an IVP.
\begin{exercise} \label{eg:de-solution-intro-ivp}
  Which of the following solves \(u'(t) = t u(t)\) with the initial condition \(u(0) = 2\)?
  \[
    f_{1}(t) = e^{t^{2}}, \qquad f_{2}(t) = 2e^{t^{2}/2}, \qquad f_{3}(t) = 3e^{t^{2}/2}.
  \]

  \blanks{15}
\end{exercise}

Pro-tip: Always check the initial condition first.
\begin{exercise}
  Which of the following is the solution to \(y(x)y'(x) = x\) with initial condition \(y(0) = -1\)? \hlwarn{Edited: added missing ``?''}
  \[
    \qquad \sqrt{2x} - 1, \qquad \sqrt{2x - 1}, \qquad -\sqrt{2x + 1}.
  \]

  \blanks{15}
\end{exercise}
\clearpage

Initial-value problems are not brand new ideas. We ran into some versions of them in Calculus~1000.
\begin{example} \label{ex:diff-eq-IVP-calc-1000}
  Consider the initial-value problem \(y' = x^{2}\) with \(y(0) = 4\).  Which \emph{one} of the following problems from Calculus~1000 asks us to solve the initial-value problem?

  \begin{enumerate}[label=(\alph*)]
    \item Find antiderivatives of \(x^{2}\).
    \item Integrate \(x^{2}\).
    \item Find the antiderivative \(F\) of \(f(x) = x^{2}\) that satisfies \(F(0) = 4\).
    \item Find the tangent line of \(y = x^{2}\) passing through the point \((0,4)\).
    \item None of the above.
  \end{enumerate}

  \blanks{20}
\end{example}
\clearpage

Like recurrences, certain initial conditions of differential equations lead to particularly simple solutions. 

\begin{definition}[equilibrium solution] \label{def:diff-eq-equilibrium}
  An \hlmain{equilibrium solution} (or equilibrium) of a differential equation is a constant function.  
\end{definition}

Notations for equilibrium solutions of differential equations vary. A common notation is to add a bar \hlwarn{Edited: ``an bar'' \(\to\) ``a bar''} to the notation of the dependent variable. For example, if \(u\) is the dependent variable of a differential equation, then it is quite common to write \(\bar{u}\) to denote an equilibrium. 

\faStar{} Equilibrium solutions are very easy to find because they are constants. Simply replace all occurrences of the \emph{dependent} variables \(u\) (or \(u(t)\)) by \(\bar{u}\) and solve for \(\bar{u}\) using basic algebra. It is possible to find none, exactly one or more than one equilibrium solution.

\begin{example} \label{eg:equilibria-logistic}
  Find all equilibria of \(N'(t) = r N(t) \left(1 - \frac{N(t)}{K}\right)\) where \(r,K\) are constants.

  \blanks{10}
\end{example}
